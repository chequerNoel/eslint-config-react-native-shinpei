# eslint-config-react-native-shinpei
My eslint configuration for React Native. this plugin requires eslint `v6.6.0` or superior.


## Usage

```
npm i eslint-config-react-native-shinpei -D
```

In `.eslintrc`:

```json
{ 
  "extends": "react-native-shinpei" 
} 
```
In `eslint.config.js`:

```javascript
module.exports = {
  extends: "react-native-shinpei",
}
```
